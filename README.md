<h1>🔰 Installation And Requirements 🔰</h1>


```bash
apt update && apt upgrade -y
```

```bash
apt install git -y
```


```bash
git clone https://github.com/Dark-Legends/linux-distro-android.git
```


```bash
cd linux-distro-android && chmod +x setup.sh
```


```bash
./setup.sh
```
